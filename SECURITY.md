# Security policy

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Use GitHub's
private vulnerability reporting for this repository instead:

https://github.com/AlmanacCode/Yoke/security/advisories/new

Include reproduction steps, affected versions, and potential impact. We will
acknowledge reports as quickly as practical and coordinate disclosure after a
fix is available.

## Supported versions

Yoke is currently alpha software. Security fixes are applied to the latest
published release.
