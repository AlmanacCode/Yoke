06 — Required product rule

Classify the instance, not only the surface

Yoke should compute effective support from provider surface × supplied representation × lowering path × live evidence.

A future report should say, for example: “Codex supports native skills; this agent has one native path-backed skill and two prompt-compiled inline skills.” Here, prompt-compiled means Yoke copies the skill's instruction text into the parent prompt. It does not create a directory, expose a native skill identity, lazy-load the body, include supporting files, or produce a provider skill invocation. The same calculation must apply to subagents, workflows, goals, hooks, plugins, MCP, models, and storage.
